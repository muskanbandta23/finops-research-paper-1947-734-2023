# A FinOps Approach to Cloud Cost Governance
### Bridging Engineering and Finance

Final year research paper submitted as part of **COOP-2 (22CS421)**, BE-CSE 8th Semester, Chitkara University.

## Abstract
This paper examines how FinOps can be implemented as a cloud cost governance framework that connects engineering, finance, and business teams. Using a mixed-methods approach (literature review, case studies of 5 organizations across tech, retail, healthcare, finance, and manufacturing, plus 18 stakeholder interviews), we propose a hub-and-spoke governance model and a 3-phase implementation roadmap aligned with the Crawl-Walk-Run maturity framework.

## Key Findings
- Organizations adopting FinOps achieved an **average cloud cost reduction of 27.4%** within the first year
- Mature (Run-stage) organizations reached **30–35% cost savings**
- Cross-functional collaboration improved resource allocation by **38–40%**
- Resource tagging coverage above 85% strongly correlated with reduced idle spend
